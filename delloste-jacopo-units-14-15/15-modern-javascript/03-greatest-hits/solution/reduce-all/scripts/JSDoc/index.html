main.js

This script implements custom versions of common array methods using `Array.prototype.reduce` and The Modern Javascript's ES6+ Syntax. The functions include: myForEach, myMap, myFilter, myIndexOf, and mySlice. The script then compares the outputs of these custom functions with the built-in JavaScript methods.
