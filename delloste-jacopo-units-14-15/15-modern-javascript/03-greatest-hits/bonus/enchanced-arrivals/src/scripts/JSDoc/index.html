helper.js

This module simulates a dynamic flight departures board. It manages a list of flights, periodically generating new entries and updating their status over time (DEPARTING, ON_TIME, DELAYED, ARRIVED). Users can view detailed flight information on hover, and toggle a popup showing currently departing flights.

main.js

Entry point for the departures board application. It imports the `arrivals` module and initializes the UI by: - Adding an event listener to the "Departures" button. - Creating new flights at regular intervals. - Updating the flight table periodically.
