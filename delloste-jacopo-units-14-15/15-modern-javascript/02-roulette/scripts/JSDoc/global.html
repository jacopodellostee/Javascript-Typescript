(async) playGame() → {Promise.<void>}

Plays three rounds of roulette concurrently. Uses `Promise.all` to execute three rounds in parallel. Logs each result if all rounds succeed. If any round is lost, the error is caught and its message is logged. In all cases, "Game over" is logged at the end.

A promise that resolves when the game is over.

Promise.<void>
Example
playGame();
// Possible outputs:
// "Round 1: won!"
// "Round 2: won!"
// "Round 3: lost!"
// "Game over"

roulette(labelopt, delayopt) → {Promise.<string>}

Simulates a roulette round with a random 50/50 outcome. After the specified delay, the promise either resolves with a win message or rejects with a loss message.
Parameters:

label string <optional>
